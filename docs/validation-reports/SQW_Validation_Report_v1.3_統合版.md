## 1. プロジェクト概要

### 1.1 目的
SQWサーベイ2は「個人の幸福×チームの成果」を同時に測定できるツール。エンゲージメントサーベイとの差別化として「成果と幸福を同時に測定できるツール」として訴求。

### 1.2 収益モデル
- B2B：チーム単位・法人契約
- B2C（新規）：AIレポートの個人販売
  - 個人レポート：¥500〜
  - チームレポート：¥2,000〜
  - チーム一括プラン：¥5,000〜
- 販売方式：Gallupストレングスファインダー方式のコード販売（SQWAI-XXXX-XXXX）
- 代理店：卸値販売、販売価格はWHI不介入

### 1.3 技術スタック
- Next.js / TypeScript / Supabase / Vercel
- Anthropic API（claude-sonnet、本番キー設定済み）
- Resend（メール配信）
- 本番URL：https://sqw-survey.vercel.app
- GitHub：https://github.com/Katsuaki-WHI/sqw-survey

### 3.2 ワゴン世界観のキーワード（設問との整合）
- 景色：誇りを持って向かえる景色か・なぜここへ向かうか
- 道筋：石畳か泥道か・道への信頼があるか
- ロープ：ロープがピンと張っているか・目標と期待がロープを通じて伝わっているか
- タイヤ：丸いか四角いか・公正な評価・自律性・前例への挑戦

- 積荷：旅の装備（仲間の地図×外部ネットワーク×最新ツール）が揃っているか
- 登場人物の多彩さ：それぞれが個性で舞台に立ち、ぶつかり合いながら化学反応を起こしているか
- はたらく喜び：旅が楽しそうか・この旅に幸せを感じているか

## 5. AIレポート設計原則

### 5.1 基本方針
- 「診断」でなく「応援」トーン
- 言葉遣い：課題→成長の余地、弱み→伸びしろ
- B2B/研究用語（Gallup Path等）はレポート本文に出さない
- 無料チラ見せ→有料完結
- CTAはカテゴリ別スコアにはなし、示唆セクション末尾に1つだけ集約

### 5.2 無料/有料レポートの構成
- 無料個人：カテゴリ別スコア＋示唆コメント
- 無料チーム：カテゴリ別スコア＋示唆コメント＋設問別スコア
- 有料個人：強み分析・誇り×幸福の4象限・アクションプラン・伝え方ヒント
- 有料チーム：輝いている状態・消耗している状態・最重要の1つ・リーダーへの示唆
